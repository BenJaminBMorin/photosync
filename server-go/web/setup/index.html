PhotoSync Setup

Let's get your photo backup server configured

Welcome to PhotoSync

This setup wizard will help you configure your PhotoSync server. You'll need:

  1. A Firebase project with Cloud Messaging enabled
  2. Your email address for the admin account
  3. The PhotoSync mobile app installed on your phone

The entire process takes about 5 minutes.
